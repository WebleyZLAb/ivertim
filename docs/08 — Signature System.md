# Purpose

A strong brand is not remembered only because of its logo.

It is remembered because every visual touchpoint feels like it belongs to the same system.

Most companies stop after defining:

- Logo
- Typography
- Colors
- Icons

Ivertim goes one step further.

It introduces a recurring visual language called the **Signature System**.

The Signature System is not a graphic style.

It is not decoration.

It is not an illustration library.

It is a visual DNA that connects every part of the brand without demanding attention.

Users should rarely notice the Signature System consciously.

Instead, they should subconsciously feel consistency between:

- website
- presentations
- social media
- proposals
- business cards
- folders
- printed materials
- signage
- advertising
- future digital products

The system exists to create recognition through repetition, not through visual noise.

---

# Design Philosophy

Everything inside the Signature System comes from one single visual element.

Nothing is invented separately.

Nothing is added only because it "looks nice".

Every application grows from the same geometric foundation.

The goal is not visual complexity.

The goal is visual consistency.

A successful Signature System should create the feeling that every element belongs to the same company, even when the logo is absent.

If the logo is removed from a design, the user should still have the feeling that it belongs to Ivertim.

That is the purpose of this system.

# Core Principle

Everything begins with one element.

That element is called the **Signature Shape**.

Every other visual expression inside the system must originate from it.

There should never be multiple independent decorative systems.

One shape.

One language.

Infinite applications.

---
# Signature Shape

## Definition

The Signature Shape is the smallest visual building block of the Ivertim identity.

It originates directly from the geometry established inside the official logo.

It is not an icon.

It is not an illustration.

It is not a decorative blob.

It is a deliberate geometric form that becomes the recurring visual signature of the brand.

Its purpose is not to attract attention.

Its purpose is to quietly connect different pieces of communication into one recognizable system.

Whenever users repeatedly encounter the Signature Shape across different media, they begin associating the feeling of familiarity with the Ivertim brand.

Recognition happens subconsciously.

That is its greatest strength.

---

## Philosophy

The Signature Shape represents completion.

Not interruption.

Not aggression.

Not ornament.

Its soft geometry reflects the philosophy already established through the brand strategy:

- precision
- competence
- responsibility
- craftsmanship
- finished details

Rather than acting as decoration, the Signature Shape symbolizes the final refined touch that distinguishes professional work from ordinary work.

It is intentionally subtle.

It never competes with content.

It supports content.

---

## Visual Behaviour

The Signature Shape should always feel:

- calm
- intentional
- balanced
- minimal
- premium

It should never feel:

- playful
- random
- decorative
- loud
- trendy

The Signature Shape is timeless.

Its purpose is to remain relevant for many years without depending on current design trends.


# Signature Pattern

## Definition

The Signature Pattern is not an independent graphic element.

It is the natural extension of the Signature Shape.

Every pattern must originate from repeating, rotating or mirroring the Signature Shape.

No additional decorative elements should be introduced.

The pattern exists to expand the visual identity while preserving complete consistency.

Its purpose is to create rhythm, texture and recognition without relying on additional graphic styles.

---

## Philosophy

The Signature Pattern should never become the visual focus.

Its role is supportive.

It should enhance hierarchy without competing with typography, photography or content.

When used correctly, users rarely notice the pattern itself.

Instead, they experience a subtle feeling that every visual element belongs to the same brand.

---

# Approved Pattern Compositions

Only predefined compositions should become part of the visual identity.

Future compositions may be added, but only if they clearly originate from the Signature Shape.

Recommended composition families include:

## Circle

Four Signature Shapes forming a complete circular composition.

Used for:

- premium backgrounds
- emboss textures
- packaging
- presentation covers

---

## Flower

Four Signature Shapes rotated outward creating a floral or radial appearance.

Used for:

- premium print
- certificates
- folders
- presentation details

---

## Wave

A repeated horizontal rhythm created by alternating Signature Shapes.

Used for:

- section dividers
- website backgrounds
- banners
- social media

---

## Grid

Repeating Signature Shapes aligned into a modular grid.

Used for:

- texture
- packaging
- large printed surfaces
- exhibition graphics

---

## Corner

A cropped composition entering from one corner.

Used for:

- hero sections
- CTA blocks
- posters
- brochure covers

---

## Linear

Repeated Signature Shapes arranged along a single direction.

Used for:

- separators
- document headers
- navigation details
- page transitions

# Applications

The Signature System should appear naturally throughout the complete visual identity.

It should never feel forced.

---

# Website

Recommended applications include:

## Photography

A single rounded Signature corner may replace a standard image corner.

Never apply it to every image.

Recommended usage:

- one image within a gallery
- one featured image
- one hero supporting image

Variation creates rhythm.

Repetition creates fatigue.

---

## Quote Marker

Instead of generic quotation graphics or vertical bars, the Signature Shape may be used as the opening visual marker.

Example:

◜

Competence begins before the first board is cut.

This immediately reinforces visual recognition without adding unnecessary decoration.

---

## CTA Elements

The Signature Shape may replace generic arrows or decorative icons.

Examples:

◜ Talk to us

◜ Start your project

◜ Request a proposal

The Shape should never compete with the CTA label.

It simply becomes part of the Ivertim language.

---

## Section Backgrounds

Large Signature Shapes may appear behind content.

Rules:

- extremely low opacity
- oversized
- partially cropped
- never fully visible
- never reduce readability

Purpose:

Create depth.

Not decoration.

---

## Section Separators

Instead of simple horizontal lines, Signature compositions may separate important sections.

The separator should always feel elegant and understated.

---

## Cards

Cards may incorporate the Signature Shape in one corner or as a subtle visual accent.

Never apply it to every card inside the same grid.

Variation is preferred.

---

## Hover Details

Very small Signature elements may appear during interaction.

Examples:

- CTA hover
- image hover
- card hover

Hover states should reinforce the identity rather than introduce new visual effects.

---

- business cards

Whenever possible, prefer physical finishing techniques instead of additional colors.

Recommended:

- Emboss
- Deboss
- Blind emboss
- Blind deboss
- Spot UV
- Soft touch materials

The Signature Pattern is particularly suitable for premium finishing because it communicates quality through texture rather than color.

# Motion & Advertising

The Signature Shape may become the primary visual element inside advertising campaigns.

Unlike interface design, advertising may use more expressive animation if it improves memorability.

Examples include:

- shapes entering sequentially
- repeated visual rhythm
- multiple Signature Shapes merging into one
- final transition into the Ivertim logo

The objective is not animation itself.

The objective is recognition.

If users begin associating a particular motion language with Ivertim before the logo appears, the Signature System has successfully extended into motion design.

Motion behaviour should always originate from the Signature Shape rather than introducing unrelated graphic effects.


# General Rules

The Signature System exists to reinforce the Ivertim identity, not to replace it.

Every application should support the content rather than compete with it.

Whenever uncertainty exists, the simpler solution is usually the correct one.

The Signature System should always feel intentional.

Never decorative.

---

## Principle 01 — Content Comes First

The user's attention should always follow this order:

1. Message
2. Content
3. Photography
4. Interface
5. Signature System

If the Signature System becomes the first thing users notice, it has been overused.

---

## Principle 02 — Less Creates More Recognition

Recognition is created through repetition.

Not quantity.

One Signature Shape placed correctly is more valuable than twenty decorative shapes.

Every application should have a clear purpose.

---

## Principle 03 — Consistency Beats Creativity

Future designers should not invent new visual languages for every campaign.

Instead, creativity should happen **inside** the Signature System.

The more consistently the same visual language is repeated, the stronger the brand becomes.

---

## Principle 04 — One Family

Every visual expression should originate from the Signature Shape.

Avoid introducing additional decorative systems unless they become official extensions of this document.

This keeps the identity clean, scalable and instantly recognizable.

---

## Principle 05 — Invisible Branding

The strongest branding often goes unnoticed.

The Signature System should create familiarity rather than demand attention.

Users should remember the experience.

Not the decoration.

# What NOT To Do

The following practices are considered incorrect and should be avoided.

---

## Do NOT decorate empty space

Never add Signature Shapes simply because an area feels visually empty.

Whitespace is intentional.

Respect it.

---

## Do NOT use the Shape everywhere

Not every page needs the Signature Shape.

Not every image needs a Signature corner.

Not every section requires a Pattern.

Selective use creates value.

Constant repetition creates fatigue.

---

## Do NOT invent random compositions

Every Pattern must originate from the official Signature Shape.

Avoid:

- abstract blobs
- unrelated geometric ornaments
- random circles
- decorative waves
- trendy gradients pretending to be part of the system

The Signature System should remain visually disciplined.

---

## Do NOT compete with typography

Typography communicates.

The Signature System supports.

If the Shape begins competing with headlines or calls-to-action, reduce or remove it.

---

## Do NOT reduce readability

The Signature System should never:

- lower contrast
- cover text
- interrupt reading flow
- reduce accessibility
- interfere with navigation

Good branding always improves usability.

Never the opposite.

---

## Do NOT overuse opacity effects

Large background Shapes should remain subtle.

Recommended opacity should generally stay very low.

The purpose is atmosphere.

Not illustration.

---

## Do NOT distort the Shape

Never:

- stretch
- squash
- skew
- freely redraw
- modify proportions

The Signature Shape is part of the identity.

Treat it with the same respect as the official logo.

---

## Do NOT rotate randomly

Rotation should only occur when defined by approved compositions.

Random rotation weakens recognition.

---

## Do NOT replace the logo

The Signature Shape supports the logo.

It never replaces it.

Users should never mistake one for the other.

---

## Do NOT follow trends

The Signature System is designed to outlive visual trends.

Avoid adapting it to:

- temporary UI trends
- fashionable gradients
- decorative effects
- seasonal aesthetics

Timeless brands evolve carefully.

They do not chase trends.

As the Ivertim brand grows, new applications may be introduced.

- icon families
- custom illustrations

- vehicle graphics

- branded environments

Any future addition must satisfy one condition:

It must clearly originate from the Signature Shape.

If this connection cannot be established, it does not belong to the Signature System.

# Design Philosophy

A strong brand is rarely remembered because of one logo.

It is remembered because every interaction feels connected.

The Signature System exists to create that connection.

Its purpose is not to decorate.

Its purpose is to quietly remind people that every proposal, every presentation, every website section, every printed document and every physical space belongs to the same company.

The Signature System should never become the hero of a composition.

The work itself should remain the hero.

The Signature System simply provides the invisible thread that ties everything together.

If people consciously notice the Signature Shape before they notice the content, the system has been overused.

If they cannot explain why everything feels consistent—but they instinctively recognize it as Ivertim—the Signature System has achieved its purpose.

---

# Final Statement

The Signature System is not a graphic asset.

It is not a pattern library.

It is not decoration.

It is a visual language.

A language that should quietly accompany the Ivertim brand for years to come.

Simple enough to remain timeless.

Flexible enough to evolve.

Consistent enough to become instantly recognizable.

That is the goal.

That is the Signature System.
